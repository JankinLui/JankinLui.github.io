DVWA搭建

DVWA安装

Wget https://github.com/ethicalhack3r/DVWA/archive/master.zip 下载dvwa压缩包

安装所需环境

[root@wgxy63 ~]# p-gd yum install -y httpd mysql-server mysql php php-mysql ph
注：php-gd 库：gd 库是 php 处理图形的扩展库，gd 库提供了一系列用来处理图片的 API，使用 GD 库可以处理图片，或者生成图片。 在网站上 GD 库通常用来生成缩略图或者用来对图片加水印或者对网站数据生成报表及验证码

全部设置开机启动，然后启动服务

测试 LAMP 环境:

[root@wgxy63 ~]# vim /var/www/html/test.php

访问 http://192.168.10.9/test.php 正常，说明 LAMP 环境支持 PHP 解析没有问题

配置 mysql 数据 root 用户密码：

[root@wgxy63 ~]# mysqladmin -u root password “123456”
注：若已经设置 mysql 数据 root 用户密码，则更改：
[root@wgxy63 ~]# mysql -u root –p687145
mysql> set password=password(‘123456’);
mysql> quit

[root@wgxy63 ~]# mysql -u root -p123456
mysql> show databases;

将下载的 dvwa 渗透系统代码上传到 Linux 上，并解压到网站根目录下
[root@wgxy63 ~]# unzip -d /var/www/html/ DVWA-1.9.zip
[root@wgxy63 ~]# ls /var/www/html/
DVWA-1.9 test.php
[root@wgxy63 ~]# chown apache:apache /var/www/html/DVWA-1.9/ -R

编辑/dvwa/config/config.inc.php,配置数据库信息，user和password是MySQL的用户名和密码。
[root@wgxy63 ~]# vim /var/www/html/DVWA-1.9/config/config.inc.php 修改的地方如下：
1 $_DVWA[ ‘db_server’ ] = ‘127.0.0.1’;
2 $_DVWA[ ‘db_database’ ] = ‘dvwa’;
3 $_DVWA[ ‘db_user’ ] = ‘root’;
$_DVWA[ ‘db_password’ ] = ‘123456’; #只需要修改成你的 mysql 的 root 用户密码

部署 DVWA 网站系统

在浏览器中输入：http://192.168.112.63/DVWA-1.9/setup.php

解决报错信息：

[root@wgxy63 ~]# vim /etc/php.ini 改：815行 allow_url_include = Off 为： allow_url_include = On
[root@wgxy63 ~]# service httpd restart

解决：

[root@wgxy63 ~]# vim /var/www/html/DVWA-1.9/config/config.inc.php 改：
4 $_DVWA[ ‘recaptcha_public_key’ ] = ‘’;
5 $_DVWA[ ‘recaptcha_private_key’ ] = ‘’;
为：
29行 $_DVWA[ ‘recaptcha_public_key’ ] = ‘6LdK7xITAAzzAAJQTfL7fu6I-0aPl8KHHieAT_yJg’;
$_DVWA[ ‘recaptcha_private_key’ ] = ‘6LdK7xITAzzAAL_uw9YXVUOPoIHPZLfw2K1n5NVQ’;

顺手把 dvwa 默认的难度给调整一下：
32 $_DVWA[ ‘default_security_level’ ] = ‘low’;

chmod 777 /var/www/html/DVWA-master/hackable/uploads
chmod 777 /var/www/html/DVWA-master/external/phpids/0.6/lib/IDS/tmp/phpids_log.txt

dvwa 的登录界面的默认用户名和密码为 admin 和 password
点 Create / Reset Database 后，等 2 秒，它会自动跳转后台管理登录页
