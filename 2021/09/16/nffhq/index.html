开关防火墙&软件包开机自启

setenforce 0 # 设置SELinux 成为permissive模式 （关闭SELinux）
setenforce 1 # 设置SELinux 成为enforcing模式 （开启SELinux）

#或者修改配置
vi /etc/selinux/config
#SELINUX=enforcing
#注释掉
#SELINUXTYPE=targeted
#注释掉
SELINUX=disabled
#增加
:wq! #保存退出
setenforce 0

或者设置SELinux

centos7

#systemctl stop firewalld –临时关闭防火墙

#systemctl is-enabled firewalld –检查防火墙是否开机启动（简化结果）

#systemctl is-active firewalld –检查防火墙是否运行（简化结果）

#systemctl disable firewalld –永久关闭防火墙

#setenforce 0 –临时关闭SELiunx

Setenforce: SELiunx is disabled

#vim /etc/seliunx/config –永久关闭seliunx

SELIUNX=disabled //将enforcing改为disabled

centos6

/etc/init.d/iptables stop //临时关闭防火墙

chkconfig iptables off //设置开机关闭防火墙

Iptables -F //关闭防火墙

软件自启

  1. 查看是否已经安装

#rpm -qa | grep httpd

httpd-2.2.15-5.el6.i686

httpd-tools-2.2.15-5.el6.i686

默认已安装httpd，跳过“2.安装”，直接到“3.启动”

  1. 安装

(1) 挂载光驱

#mount /dev/cdrom /media

mount: block device /dev/sr0 is write-protected, mounting read-only

#cd /media/Packages

#ls httpd*

httpd-2.2.15-5.el6.i686.rpm

(2) 安装

#rpm -ivh httpd-2.2.15-5.el6.i686.rpm

  1. 启动

#service httpd start //启动

  1. 关闭防火墙

#service iptables stop //关闭防火墙

#chkconfig iptables off //设置开机关闭防火墙

#chkconfig –list | grep iptables //检查开机状态

iptables 0:关闭 1:关闭 2:关闭 3:关闭 4:关闭 5:关闭 6:关闭

  1. 关闭SELinux

#setenforce 0 //设置SELinux为Permissive

#getenforce //查看状态

Permissive //宽容模式

#vim /etc/selinux/config //设置开机SELinux为Permissive

SELINUX=Permissive

#getenforce

Permissive

  1. 开机自动启动

#chkconfig httpd on //设置开机自动启动

#chkconfig –list | grep httpd //检查开机状态

httpd 0:关闭 1:关闭 2:启用 3:启用 4:启用 5:启用 6:关闭

            0关机  1单用户 2字符界面 3字符界面  4保留  5图形界面 6重启

            2-5启用！

#chkconfig iptables off //设置开机关闭防火墙

#chkconfig –list | grep iptables //检查开机状态

iptables 0:关闭 1:关闭 2:关闭 3:关闭 4:关闭 5:关闭 6:关闭

#vim /etc/selinux/config //设置开机SELinux为Permissive

SELINUX=Permissive

  2. 测试

ping通（网络层、传输层），Web服务通（应用层）！配置成功！
