1.2 配置本地YUM源

yum的一切配置信息都储存在一个叫yum.repos.d的配置文件中，通常位于/etc/yum.repos.d目录下

备份yum.repos.d下的所有文件，例如在文件后加入.bak进行备份

如图

创建一个新的yum源配置文件，yum源配置文件的结尾必须是.repo
[root@xuegod63 yum.repos.d]# vim CentOS7.repo #写入以下红色内容
[CentOS7]
name=CentOS-server
baseurl=file:///mnt
enabled=1
gpgcheck=0
参数说明：
[CentOS7] —>yum的ID，必须唯一
name=CentOS-server —–>描述信息
baseurl=file:///mnt ——-> /mnt表示的是光盘的挂载点 . file:后面有3个///
enabled=1 ——>启用
gpgcheck=0 —->取消验证

清空并生成缓存列表
[root@localhost ~]# yum clean all #清空yum缓存
[root@localhost ~]# yum list #生成缓存列表
验证一下
[root@xuegod63 yum.repos.d]# yum -y install vim

1.3 更改网络yum源

mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.backup 
wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo
mv /etc/yum.repos.d/epel.repo /etc/yum.repos.d/epel.repo.backup
mv /etc/yum.repos.d/epel-testing.repo /etc/yum.repos.d/epel-testing.repo.backup
wget -O /etc/yum.repos.d/epel.repo http://mirrors.aliyun.com/repo/epel-7.repo

ps:yum repolist 列出yum源信息；

第二章 网络配置静态

/etc/sysconfig/network-scripts/ifcfg-eno16777728为网卡配置文件，后面的数字为查询ip add命令显示的你要更改的有ip的网卡

重要文件–网卡配置文件
TYPE=Ethernet –指定网络类型 以太网Eehernet 电话10M 军用（帧中继） 金融公司（FastEthernet）
BOOTPROTO=none –网络启动协议 如何让主机得到IP地址
a. 自己手动配置 none static （静态）
b. 自动获取地址 dhcp （动态）
NAME=eth0 –主机网卡的名称 逻辑名称
UUID=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx –虚拟主机的硬件标识
DEVICE=eth0 –主机网卡的名称 设备名称物理名称
ONBOOT=yes –设置网卡是否处于开启状态（激活状态）
IPADDR=xxx.xxx.xxx.xxx –静态ip地址
PREFIX=24 –子网掩码
GATEWAY=XXX.XXX.XXX.XXX –网关
DNS1=XXX.XXX.XXX.XXX

网关
